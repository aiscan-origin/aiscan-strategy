
[AIScan 전략 포트폴리오]

■ 프로젝트 개요
- 프로젝트명: AIScan
- 기획자: aiscan-origin
- 최초 공개: GitHub (2025년 5월)
- 목표:
  • 네이버 검색 기반의 LLM 설계
  • 한국어 신뢰도 높은 응답 시스템 구축
  • 검색 + 대화 통합형 플랫폼 전략 수립

■ 핵심 설계 구조 (한 줄 요약)
"검색을 LLM의 뇌로 전환하여, 구조화된 실시간 응답 시스템을 설계한다."

■ 기술 흐름 단계
1단계: 사용자 질의 입력
2단계: 실시간 검색 API 호출 (네이버 검색, 지식인, 사전, 지도 등)
3단계: 정보 구조화 및 요약 정제
4단계: LLM 입력 → 감정 UX 포함 응답 생성

■ 차별 포인트
- 기존 GPT 구조와 달리, 검색이 응답 흐름을 주도함
- BingChat과는 반대로, 자산 구조 최적화 및 감정 응답 포함
- 네이버의 모든 데이터 자산을 하나의 LLM 구조로 통합
