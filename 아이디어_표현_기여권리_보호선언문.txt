
[아이디어·표현·기여권리 보호 선언문 – AIScan 프로젝트]

────────────────────────────
■ 선언자: GitHub 사용자명 `aiscan-origin`
■ 프로젝트명: AIScan – 네이버 검색 기반 LLM 구조 설계 프로젝트
■ 최초 기획일: 2025년 5월

────────────────────────────
■ 선언 내용:

본 프로젝트에 포함된 전략 설계, 구조 흐름, 문서 구성, 명칭(AIScan)은  
2025년 5월 `aiscan-origin`에 의해 최초 기획 및 기록되었으며,  
다음 사항에 대해 권리 보호를 명시합니다.

1. AIScan이라는 명칭 및 LLM 구조에 대한 전략 프레임은 고유의 창작물임
2. GitHub와 Notion, GPT 기반 대화기록을 통해 최초성 및 작성자가 입증됨
3. 공개 후 동일한 구조나 표현이 타인에 의해 사용될 경우,  
   본 문서와 GitHub 커밋 타임스탬프를 통해 우선 기여권 주장 가능
4. 제3자가 상업적, 정책적, 기술적으로 해당 구조를 활용하고자 할 경우  
   본 전략 설계자에게 출처 명시 및 협의가 필요함

────────────────────────────
■ 활용 목적:

- 창작물의 기여권 보호
- 전략적 명칭 및 구조의 도용 방지
- 공익적 확산은 환영하되, 기여자 표시는 반드시 요구
- 정책 및 기업 연계 시 최초 기획자에 대한 명예·공로 보장

────────────────────────────
■ 기여자 확인:

해당 구조는 GPT 기반 논의와 사용자의 직접 입력으로 생성되었으며,  
2025년 5월 GitHub 저장소 `aiscan-origin/aiscan-strategy`를 통해  
전 세계에 최초 공개되었습니다.
